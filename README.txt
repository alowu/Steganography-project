Приложение встраивания и извлечения сообщений методом Куттера. Размерность креста для прогнозирования 5х5 пикселей.
Чтобы встроить сообщение в изображение:
1. Откройте изображение с помощью кнопки 'Open File' в левой части приложения.
2. Введите сообщение в текстовое поле под изображением.
3. Ниже введите ключ для генерации псевдослучайной последовательности пикселей, в которые будет встраиваться сообщение, и значение энергии встраивания (значение по умолчанию 0.5).
4. Нажмите кнопку 'Encrypt'. В текстовом поле 'Key for decryption:' будет выведен ключ для обратного извлечения изображения. В правой части приложения появится изображение со встроенным сообщением. Это изображение можно сохранить нажав на кнопку 'Save File'.
Для извлечения сообщения из изображения используйте изображение, посмещённое в правую чать приложения после встравания, или нажав на кнопку 'Open File' выберите изображение для извлечения сообщения.
В поле 'Enter key for decryption:' введите ключ для извлечения изображения.
Если у вас нет ключа - вы не сможете извлечь сообщение! https://github.com/alowu/Steganography-project
